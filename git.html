Git local commit:

* Open git gui
* Select all files that are need to be push
* Write comment
* Commit
* Check log

Example

git gui
git log --oneline -10


Git cherry-pick:

Before push you need to
git log --oneline -10
git log --oneline -10 origin/branch_name
git push origin HEAD:refs/for/branch_name

Example

git log --oneline -10
git log --oneline -10 origin/development
git push origin HEAD:refs/for/development

If conflict, then resolve that conflict and commit your changes again.



Git push:

Check both log. If you have latest code in both branch only then push. Otherwise get the latest code.
git log --oneline -10
git log --oneline -10 origin/branch_name
git push origin HEAD:refs/for/branch_name

Example

git log --oneline -10
git log --oneline -10 origin/development
git push origin HEAD:refs/for/development


Git conflict resolve:

* Resolve those conflicted code ("Keep both changes" means - your code + origin code, "Incoming changes" means - changes you made / your code, "Current changes" means - origin code )
* Open git gui
* Amend last commit
* Select all files (conflict resolved files) that are need to be push
* Commit your code locally

Example

git gui
git log --oneline -10


Git merge 2 branch:

Example

This Example shows how to push your changes step by step.

Example

Commit code locally.
git fetch origin
git log --oneline -5
git log origin/development --oneline -5
git checkout origin/development
git checkout -b new_branch
git cherry-pick my_sha1
If conflict, resolve it, amend last commit and commit again check:
git log --oneline -5
git log origin/development --oneline -5
git push origin HEAD:refs/for/development
npm start
To get Latest code:
git log --oneline -5
git log origin/development --oneline -5
git fetch origin
git log --oneline -5
git log origin/development --oneline -5
git checkout origin/development
git checkout -b new_branch
git fetch
git checkout origin/development
git checkout -b new_branch_name
If you miss any file while push then:
select those file
Commit again
git push origin HEAD:refs/changes/56926
56926 = your last push id
